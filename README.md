# Chapterly
Chapterly is a modern figma design of a book reading app. 
It is a modern, user-friendly mobile application design which is designed with Figma for passionate book lovers who want to explore, organize, and enjoy reading in a seamless way. Whether you prefer e-books or audiobooks, Chapterly brings all your reading needs into one place, making the experience engaging, personalized, and motivating.
